
IOTVerse is a full-stack social networking web application that allows users to connect, share content, and interact in a simple and secure environment. The project was developed to demonstrate core full-stack development skills, including backend logic, RESTful APIs, database management, and responsive front-end design without the use of modern JavaScript frameworks.


Features

- User registration and login
- Session-based authentication
- Create and view posts
- Follow other users
- Search for users and content
- Responsive user interface
- Secure backend API structure

Tech Stack

Frontend
- HTML
- CSS
- JavaScript (Vanilla)

Backend
- Node.js
- Express.js

Database
- MongoDB

Installation & Setup

1. Clone the repository:
2. Navigate into the project directory:

cd iotverse

3. Install dependencies:

npm install

4. Start the server:

node server.js

5. Open the application in your browser:

[http://localhost:8000](http://localhost:8000)


Key Learning Outcomes

- Implementing RESTful APIs using Express
- Handling authentication and user sessions
- Working with MongoDB for data persistence
- Structuring a full-stack application
- Building responsive UIs with vanilla JavaScript
- Debugging and testing real-world web applications


Notes
